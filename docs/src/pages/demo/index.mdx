# QRTuber In-Page Video Demo

This video can be used for testing VRTuber Setups.

To test:

- Start Intiface Central server, scan for device in devices tab
  - At the moment, device needs to be capable of vibration
- Once device is found, hit "Connect to Intiface" below
- Hit "Start Tracking"
- Start video playing

Your toy should start vibrating as the QR Code starts changing on the video.
